LocalWorkspaceRoot      F:\ISJ\Development\git\
GetLocalWorkspacePath   F:\ISJ\Development\git\ + ProjectName + src?


前提:
git init済みのvscodeライクなワークディレクトリが存在することを前提とする
典型例は
ProjectName:Project1
F:\ISJ\Development\git\Project1  ここに.gitフォルダがある(git init)
F:\ISJ\Development\git\Project1\src  ここにIRIS関連ファイルがある


Studioオープン時、NS切り替え時、Project開き直し時に、git initされてるかチェックしてされてなければ(かつCreateRepoがONなら)するように変更。

Commit命令は下記を実行
git -c user.name=$USERNAME -c user.email='$USERNAME@localhost' commit -m "auto commited"
